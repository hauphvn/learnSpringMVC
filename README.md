# learnSpringMVC
1.   **Fron controller:** tất cả request được gửi đến từ người dùng đều qua controller này.
2.   **Web service:** là một hệ thống để hỗ trợ máy tương tác với máy, hoặc component với nhau qua một hệ thống LAN, Internet. Có 2 loại: *SOAP* v
à *RESTful* . Trong nội dung này mình sẽ sử dụng Json và RESTful web service.
